This is a sipmle frontend photo gallery project developed with react. Axios was used to perform http requests.
You can clone, use and change this project.

## Dependecies

axios,
bootstrap,
font-awesome and 
react-router-dom

```
npm i axios
npm i bootstrap
npm i font-awesome
npm i react-router-dom
```
